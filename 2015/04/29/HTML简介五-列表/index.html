HTML简介五

  • 列表

  • 无序列表

< ul>
< li>< /li>
< li>< /li>
< /ul>

  • 有序列表

< ol>
< li>< /li>
< li>< /li>
< /ol>

  • 定义列表

< dl>
< dt>标题1< /dt>
< dd>选项1< /dd>
< dd>选项2< /dd>
< dt>标题2< /dt>
< dd>选项1< /dd>
< dd>选项2< /dd>
< /dl>

  • 表格

  • 格式：

< table>

< tr> 一列

< td>< /td> 一行

< td>< /td> 二行

< /tr>

< tr> 二列

< td>< /td> 一行

< td>< /td> 二行

< /tr>

< /table>

  • 合并单元格：

跨列（横向合并）：< td cospan=列数>。。。< /td>

跨行（纵向合并）: < tr rowspan=行数>。。。< /tr>

注意：合并多少，需在下一列或行减去对应的行数或列数

  • 高级

  • 标题 < caption>…< /caption>

  • 表头 < th>..< /th>

  • 页眉 < thread>…< /thread>

  • 页脚 < tfoot>..< /tfoot>
