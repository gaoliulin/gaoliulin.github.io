CSS一 字体 背景

什么是CSS？

CSS 指层叠样式表 (Cascading Style Sheets)

CSS通常称为CSS样式表或层叠样式表（级联样式表），主要用于设置HTML页面中的文本内容（字体、大小、对齐方式等）、图片的外形（宽高、边框样式、边距等）以及版面的布局等外观显示样式。

语法结构

选择器{

    属性:值;
    属性:值;
    ...
}

|   文本相关属性   |
|color|前景色（文字颜色）【 表示方式见底部备注信息】
 |font-style    |normal 正常显示  |   italic  斜体显示  |  oblique 文字倾斜
|font-weight|normal 正常显示|bold 加粗| bolder 超粗 | lighter 超细| 400  正常  |  700 加粗
|font-family|    文字字体    【字体见底部备注信息】
|font-size    |文字大小
|font:font-style  | font-weight  font-size/line-height | font-family
|line-height|
|font-family|

可以连写 font:font-style font-weight font-size/line-height font-family;

其中1,2,3,连写是可以省略.
