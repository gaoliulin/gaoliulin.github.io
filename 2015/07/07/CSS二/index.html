CSS二 CSS选择器及三大特性

CSS基础选择器

1：.class 类选择器

2：#id id选择器

3：ele 标签选择器

4：* 通配符选择器

CSS符合选择器

1：ele1 ele2 后代选择器

后代选择器用来选择元素或元素组的后代，其写法就是把外层标记写在前面，内层标记写在后面，中间用空格分隔。当标记发生嵌套时，内层标记就成为外层标记的后代。

2：ele1>ele2 子代选择器

3：ele1,ele2 并集选择器

并集选择器是各个选择器通过逗号连接而成的，任何形式的选择器（包括标记选择器、class类选择器id选择器等），都可以作为并集选择器的一部分。如果某些选择器定义的样式完全相同，或部分相同，就可以利用并集选择器为它们定义相同的CSS样式

4：p.class 交集选择器

交集选择器由两个选择器构成，其中第一个为标记选择器，第二个为class选择器或id选择器，两个选择器之间不能有空格.

伪类选择器

:link 未访问前 IE6不兼容，解决此问题，直接使用a标签。

:visited 访问后 在IE6只能应用于a连接，IE7+所有元素都兼容。

:hover 悬浮

:active 点击为松手/ 被选择的链接

:focus 伪类将应用于拥有键盘输入焦点的元素。

1. 为了产生预期的效果，在 CSS 定义中，a:hover 必须位于 a:link 和 a:visited 之后！！

2. 为了产生预期的效果，在 CSS 定义中，a:active 必须位于 a:hover 之后！！
3. Pseudo-class（伪类）的名称对大小写不敏感。

css三大特性

1.继承性

eg: tetx-  font-  line-  color  都会被后代继承
并不是所有的CSS属性都可以继承，例如，下面的属性就不具有继承性：边框、外边距、内边距、背景、定位、元素宽高属性。
所有盒子相关的属性都不能继承。   a标签  h1 标签不会实现继承

2.层叠性

是指后面设置的属性会把前面设置的属性层叠掉

3.优先级

给元素设置属性时,是有优先级的
也就是说靠近元素的样式具有最大的优先级，或者说排在最后的样式优先级最大。
!importent > 行内样式 > id选址器 > 类选择器 > 标签选择器 > 通配符 > 继承
