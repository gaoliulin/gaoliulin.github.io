js-DOM 事件 - 节点访问关系 - 元素操作

JavaScript 三个组成部分

ECMAScript: 核心（ECMAScript）欧洲计算机制造商协会,描述了JS的语法和基本对象。

DOM: 文档对象模型（DOM）,处理网页内容的方法和接口

BOM: 浏览器对象模型（BOM）,与浏览器交互的方法和接口

  • 事件

    JS是以事件驱动为核心的一门语言。

    事件三要素

    事件源、事件、事件驱动程序。

    三句话：获取事件源、绑定事件、书写事件驱动程序。
    |获取事件源：|document.getElementById(“box”);|

    |绑定事件： |box.onclick = function(){ 程序 };
    |书写事件驱动程序：|就是有关DOM的操作。

获取事件源(选择器)

|document.getElementById("box");|获取的是一个DOM元素|
|document.getElementsByTagName("box");|获取的是一个DOM元素数组|
|document.getElementsByClassName("box");|获取的是一个DOM元素数组(有兼容性问题,IE9以下的版本却不支持)|
|document.querySelector("")|获取的是一个DOM元素|
|document.querySelectorAll("")|获取的是一个DOM元素数组|

事件

|onclick    |鼠标单击|
|ondblclick    |鼠标双击|
|onkeyup    |按下并释放键盘上的一个键时触发 |
|onchange     |文本内容或下拉菜单中的选项发生改变|
|onfocus     |获得焦点，表示文本框等获得鼠标光标。|
|onblur     |失去焦点，表示文本框等失去鼠标光标。|
|onmouseover /onmouseenter     |鼠标悬停，即鼠标停留在图片等的上方|
|onmouseout /onmouseleave    |鼠标移出，即离开图片等所在的区域|
|onload    |网页文档加载事件|
|onunload    |关闭网页时|
|onsubmit    |表单提交事件|
|onreset    |重置表单时|

事件绑定

事件绑定粗略来讲三种方法:

1.直接为onlcick赋值匿名函数
div.onclick = function () {alert(1); }
2.赋值函数名;
div.onclick = fn2;//函数名就代表整个函数!
3.直接为onlcick赋值匿名函数 div.onclick = function () {alert(1); }

注意

1.事件会出现覆盖;相同的事件源和事件绑定两次,后面的会把前面的覆盖掉;
2.二次绑定,原来的程序会被覆盖;
3.绑定事件也可以用[];
    var str = 'bbb';
    (1).属性绑定1: aaa.str;      把属性绑死了,不能随意更换;(简单,快捷)
    (2).属性绑定2: aaa[str];   可以使用变量进行绑定,变量修改,实行就跟随修改;(灵活)

节点访问关系(4种)

节点(元素)操作(4种)

    -2. 父元素.insertBefore(子元素,参考元素);//把子元素添加到父元素的参考元素之前；
    

- 3.元素的删除（1种）
    父元素.removeChild(子元素);//把父元素的子元素删除
    

- 4.元素的复制（1种）
    父元素.cloneNode(boolean);//复制一份元素节点；是否复制该元素的子节点取决于boolean；
    true: 复制节点及子节点
    false: 复制节点本身不复制子节点
    注意；无法复制事件；
