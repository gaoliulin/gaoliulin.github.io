jQuery

jQuery01 jQuery简介

  • https://jquery.com/ 官网
    http://www.runoob.com/jquery/jquery-tutorial.html 菜鸟教程
    https://developer.mozilla.org/zh-CN/ 开发者网络CDN

    jQuery 是一个 JavaScript 库。

    jQuery 提供了操作DOM的方法,相比原生JS极大地简化了 JavaScript 编程。
    并且兼容于所有主流浏览器, 包括Internet Explorer 6!
    如果需要支持 IE6/7/8，那么请选择1.9

jQuery的下载：

1. 从 jquery.com 下载 jQuery 库
2. 从 CDN 中载入 jQuery
3. NPM 中载入 jQuery

jQuery库包含以下功能：

HTML 元素选取
HTML 元素操作
CSS 操作
HTML 事件函数
JavaScript 特效和动画
HTML DOM 遍历和修改
AJAX
  • 1.隐式迭代；(一把一把的操作不在需要for,而是直接操作)
    2.连式编程；(可以一直调用下去)($(“button”).click().css().hide()……;)
    3.不一样的参数,功能不一样；
  • 注意:
    jquery中的: jQuery和$一模一样;两个方法名,同一个方法;
