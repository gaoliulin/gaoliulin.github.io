c3-颜色-文本阴影-盒模型-圆角

颜色： opacity：；0-1 变透明度 子元素也会继承父元素的透明度

background: transparent;    设置完全透明的延时  子元素不会被继承

透明：RGBA

16进制 10  ff---->255

R 红色的色值  G 绿色的色值  B 蓝色的色值  A 透明度
background: rgba(255,0,0,0.5);    

HSLA(H,S,L,A)
通过hsla我们可以对颜色的透明度进行控制
background-color: hsla(240, 100%, 40%, 0.6);

取值：
H：
Hue(色调)。0(或360)表示红色，120表示绿色，240表示蓝色，也可取其他数值来指定颜色。取值为：0 - 360
S：
Saturation(饱和度)。取值为：0.0% - 100.0%
L：
Lightness(亮度)。取值为：0.0% - 100.0%
A：
Alpha透明度。取值0~1之间。

阴影

文本阴影：
text-shadow 四个参数
第一个参数  x轴方向的偏移
第二个参数  y轴方向的偏移
第三个参数  阴影的模糊度  大于等0
第四个参数  阴影的颜色
*/

c3-选择器

  1. jquery 属性选择器

    • jquery [attribute] 表示存在attribute 即可
    • jquery [attribute=value] attribute完全等于value
    • jquery [attribute*=value] attribute包含value
    • jquery [attribute^=value] attribute以value开始
    • jquery [attribute$=value] attribute以value结束

  2. 伪类选择器
    2.1 结构伪类

    + e:first-child
+ e:last-child
+ e:nth-child(n)
+ e:nth-last-child(n)
+ e:nth-child(3n+1)     注: n>=0,n必须在前面
= e.empty  选择没有子节点的e 元素

    2.2 目标伪类

    e:target  结合锚节点使用,处于当前锚点的元素会使用

eg: h2:target {color: red;}

    2.2 排除伪类

    .text:not(.special){
    color:red;
}

  3. 伪元素选择器
    e::first-letter 文本的第一个单词或字
    e::first-line 文本的第一行
    e::before e::after 是一个行内元素,需要转化成块元素

    e::selection 可改变选择文本的样式

  • 注:伪类与伪元素区别:
    伪类的效果可以通过添加一个实际的类来达到;而伪元素需要添加一个实际的元素才可以达到.

h5-storage 本地存储

本地存储参考

cookie参考

简介:

Storage模块管理应用本地数据存储，用于应用数据的保存和读取，应用本地数据localStorage,sessionStorage的区别在于数据的有效域不同，前者可在应用内跨域操作，数据存储期是持久化的，并且没有容量的限制，通过plus.storage可获取应用本地数据管理对象
  1. localstorage 一直保存在浏览器客户端
  2. sessionstorage 当浏览器关闭时就失效了
  3. cookie 在浏览器和服务器之间传递,在有效期内存在客户端(注意后台使用)

操作方法
localstorage.setItem(‘name’, ‘小高’); 存数据

localstorage.getItem('name');  取数据 
localstorage.removeItem('name')  删除数据
localstorage.clear()  清空数据

sessionstorage 同上四种
